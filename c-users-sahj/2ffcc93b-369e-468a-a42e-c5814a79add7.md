### Estructura de carpetas (PARA)
```
ObsidianVault/
├── 1-Projects/        ← Proyectos activos con deadline
├── 2-Areas/           ← Áreas de responsabilidad continua
├── 3-Resources/       ← Material de referencia

├── Dailies/           ← Notas diarias (auto-generadas)
├── Meetings/          ← Notas de reuniones
├── Templates/         ← Plantillas
│   ├── Daily Note.md
│   ├── Meeting - Standup.md
│   ├── Meeting - 1on1.md
│   ├── Meeting - Sprint Planning.md
│   └── Meeting - Retrospective.md
├── Home.md            ← Página principal / MOC
└── .obsidian/         ← Configuración
```

### Cómo usarlo
1. **Abre Obsidian** y selecciona "Open folder as vault" → `C:\Users\sahj\Documents\ObsidianVault`
2. **Daily Note**: Click en el icono de calendario (sidebar izquierdo) o `Ctrl+Alt+D` para crear la nota del día con tu agenda y tasks
3. **Nueva reunión**: Crea una nota en `Meetings/`, luego usa `Ctrl+T` → selecciona el template de la reunión que necesites
4. **Home.md**: Configúralo como tu página de inicio en Settings → Core Plugins → Home

### Plugins recomendados para instalar
- **Calendar** - Vista de calendario para navegar daily notes
- **Dataview** - Ya incluí una query en el daily template para listar reuniones del día
- **Tasks** - Gestión avanzada de tareas con fechas y recurrencia
